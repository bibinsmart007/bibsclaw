# Contributing to BibsClaw

We welcome contributions! Here is how to get started.

## Development Setup

```bash
git clone https://github.com/bibinsmart007/bibsclaw.git
cd bibsclaw
npm install
cp .env.example .env
npm run dev
```

## Guidelines
- Fork the repo and create a feature branch
- Write tests for new features
- Follow the existing code style (ESLint + Prettier)
- Keep PRs focused and small
- Update documentation for new features

## Pull Request Process
1. Update ROADMAP.md if applicable
2. Ensure CI passes (lint, test, build)
3. Request review from maintainers
4. Squash commits before merge

## Code of Conduct
Be respectful and constructive. We are building something great together.

## Questions?
Open a GitHub Issue or reach out on the discussion board.
