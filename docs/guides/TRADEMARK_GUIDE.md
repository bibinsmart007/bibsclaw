# BibsClaw Brand/Trademark Registration

## Steps to Register
1. Search USPTO TESS: tmsearch.uspto.gov
2. File application at trademark.uspto.gov
3. Class: IC 042 (Software as a Service)
4. Description: AI assistant software
5. Filing fee: ~$250-350 per class

## UAE Registration
- Ministry of Economy: economy.gov.ae
- Required: Logo, description, passport copy
- Duration: 6-12 months
