# BibsClaw Pricing Strategy

## Tiers

### Free (Starter)
- 1 AI model (Perplexity)
- 50 messages/day
- Basic voice (browser STT)
- 5 scheduled tasks
- Community support via GitHub
- Goal: User acquisition, product feedback

### Pro ($19/month)
- All 5 AI models with smart routing
- Unlimited messages
- Advanced voice + wake word
- Unlimited tasks & webhooks
- Social media automation
- Priority email support
- Goal: Core revenue, power users

### Enterprise (Custom)
- Everything in Pro
- SSO/SAML, audit trails
- Custom model fine-tuning
- On-premise deployment
- SLA guarantees (99.99%)
- Dedicated support + Slack channel
- Goal: Large orgs, high-value contracts
