# Welcome to BibsClaw

BibsClaw is a personal AI assistant featuring multi-model intelligence, voice interface, autonomous coding agent, task automation, and multi-platform access.

## Quick Start

```bash
git clone https://github.com/bibinsmart007/bibsclaw.git
cd bibsclaw
npm install
cp .env.example .env  # Add your API keys
npm run dev
```

## Key Features

- **Multi-Model AI** - Perplexity, Claude, Gemini, GPT-4o, Ollama
- **Voice Interface** - Wake word, multi-language, streaming TTS
- **Coding Agent** - 30+ tools for autonomous development
- **Task Scheduler** - Cron, webhooks, event-driven automation
- **Multi-Platform** - Web, Telegram, Slack, Discord, CLI, Mobile
