# Introducing BibsClaw: Your AI Assistant with Real-Time Search

## The Problem
Most AI chatbots are limited to their training data cutoff.

## Our Solution
BibsClaw integrates Perplexity AI for real-time web search,
accessible via Telegram bot, web dashboard, and voice interface.

## Key Features
- Real-time web search via Perplexity
- Telegram bot for mobile access
- Voice interface for hands-free use
- Task automation and scheduling
- Admin dashboard with analytics

## Tech Stack
- TypeScript, Node.js backend
- Telegraf for Telegram integration
- Railway for deployment
- Vitest for testing, Playwright for E2E
