# BibsClaw - Product Hunt Launch

## Tagline
Your personal AI assistant with multi-model brain, voice, and automation superpowers.

## Description
BibsClaw is an open-source personal AI assistant that combines 5 AI models (Perplexity, Claude, Gemini, GPT-4o, Ollama) with voice interface, autonomous coding agent, task automation, and multi-platform access.

## Key Highlights
- Smart model routing picks the best AI for each task
- Voice-first with wake word detection and 4 languages
- Autonomous coding agent with 30+ tools
- Schedule tasks with cron, webhooks, and event triggers
- Access from web, Telegram, Slack, Discord, or CLI
- Self-hostable and privacy-respecting
